# Kompozitsiya: Avtomobil + Dvigatel 

Bu loyiha Python dasturlash tilida **Obyektlar Kompozitsiyasi** prinsipini o'rganish uchun yaratilgan.

## Loyiha maqsadi
Avtomobil va Dvigatel (Motor) klasslari misolida kompozitsiya orqali qanday qilib real obyektlar modellashtirilishini amaliy ko'rsatish.

## Qanday ishlatishingiz mumkin?

1. Loyihani ko'chirib oling:
   ```bash
   git clone https://github.com/oyatulloh765/loyiha_ishi.git
   ```
2. Loyiha papkasiga kiring:
   ```bash
   cd loyiha_ishi
   ```
3. Dasturni ishga tushiring:
   ```bash
   python Loyiha1.py
   ```

## Texnologiyalar
*   **Python 3.9**
*   OOP (Object Oriented Programming) tamoyillari
